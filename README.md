# RhythmLens

**Мобильное приложение для оцифровки и анализа ЭКГ с бумажных носителей с использованием ИИ**

Кросс-платформенное Android-приложение на Kotlin + Jetpack Compose, которое позволяет:
- Сфотографировать бумажную ЭКГ камерой смартфона
- Автоматически оцифровать сигнал (12 отведений)
- Выполнить коррекцию перспективы и масштаба
- Сохранять историю, сравнивать записи и отслеживать динамику
- Генерировать реалистичное изображение бумажной ЭКГ обратно

## Технологический стек

- **Язык**: Kotlin 100%
- **Архитектура**: Clean Architecture + MVI
- **UI**: Jetpack Compose + Material 3
- **DI**: Hilt
- **Локальная БД**: Room
- **Сеть**: Retrofit + Kotlin Serialization
- **Камера**: CameraX Compose
- **Графики**: Vico Compose
- **Навигация**: Type-safe Navigation Compose
- **Оффлайн**: Полная поддержка

## Структура проекта
├── di/              ← Hilt модули
├── domain/          ← Бизнес-логика (независим от Android)
├── data/            ← Room + Retrofit
├── ui/              ← Compose-экраны (feature-first)
└── util/            ← Вспомогательные классы

## Как запустить

1. Клонировать репозиторий
2. Открыть в Android Studio (Hedgehog или новее)
3. Запустить на эмуляторе/устройстве Android 8.0+
